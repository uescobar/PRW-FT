Este es un ejemplo del uso de los atributos id y class.

Ejemplo de uso del atributo id.

El elemento <id> permite asignar un nombre único a un elemento de HTML.

Puede diferenciar a un elemento de otros.

Ejemplo de uso del atributo class.

El atributo class puede asignar un identificador a más de un elemento de HTML.
Esto es útil para.

  • Homogenizar los atributos de grupos de elementos en un documento.
  • Permitir recopilar imformación de dichos elementos a partir del código.

El atributo class puede ser utilizado incluso en elementos de tipo distinto.
